# Quest 3: Sorting Ceremony

## Story Introduction

Every student must complete the Sorting Ceremony! Create a multi-step wizard form that guides students through house selection, familiar choice, and wand selection.

## Objective

Build a 3-step form wizard with navigation and progress tracking.

## Requirements

- Step 1: Enter name, choose house (Liondudes/Scarybird/Huftybadger/Snakeyguys)
- Step 2: Select familiar (owl/cat/toad)
- Step 3: Pick wand type (oak/phoenix/dragon)
- Next/Back buttons for navigation
- Progress indicator showing current step
- Summary page showing all selections
- Cannot proceed with empty fields

## Acceptance Criteria

- [ ] Three distinct steps implemented
- [ ] Step state tracked (1, 2, or 3)
- [ ] Next/Back buttons work correctly
- [ ] Progress indicator updates
- [ ] Validation prevents empty submissions
- [ ] Final summary shows all data

## Bonus

- Add step validation
- Save progress to localStorage
- Animated transitions between steps
