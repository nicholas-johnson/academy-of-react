# Quest 3: Battle Resources

Create BattleContext tracking: global mana pool, available reinforcements, threat level (1-10). Multiple components can spend/restore mana. Warning indicators when resources low.

## Requirements (TypeScript)
- BattleContext with resource state
- Functions to modify resources
- useBattleResources custom hook
- 3+ components consume context
- Visual warnings when low
- TypeScript interfaces

## Acceptance Criteria
- [ ] Context manages all resources
- [ ] Functions update resources correctly
- [ ] Multiple components interact
- [ ] Warnings display when low
- [ ] Proper TypeScript typing
- [ ] No prop drilling
