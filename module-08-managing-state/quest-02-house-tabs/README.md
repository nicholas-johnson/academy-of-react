# Quest 2: House Tabs

Create tabbed interface showing 4 houses. Each tab shows that house's students, total points, recent achievements. Parent component manages active tab state.

## Requirements
- Four tab buttons (one per house)
- Active tab indicator
- Each tab shows different content
- Parent manages which tab is active
- Student data for each house

## Acceptance Criteria
- [ ] Four tabs render
- [ ] Clicking switches tabs
- [ ] Active tab highlighted
- [ ] Correct content shows
- [ ] State lifted to parent
