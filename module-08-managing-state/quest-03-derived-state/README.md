# Quest 3: Creature Gallery

Display 20 magical creatures students can summon in battle. Filters: type (beast/spirit/elemental), power level, mana cost. Select creatures for battle team (max 5). Display team total power.

## Requirements
- 20+ creatures with stats
- Multiple filters
- Click to add/remove from team
- Max 5 creatures on team
- Show team total power
- Visual indication of selected

## Acceptance Criteria
- [ ] 20+ creatures displayed
- [ ] Filters work correctly
- [ ] Selection toggles
- [ ] Max 5 enforced
- [ ] Total power calculates
- [ ] Selected creatures highlighted
