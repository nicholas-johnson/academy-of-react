# Quest 1: Resource Dashboard

Track academy resources: gold, mana crystals, potion supplies. Actions: "Train Student" (costs gold, increases power), "Brew Potion" (costs supplies), "Gather Resources" (increases resources). Show totals and student army power.

## Requirements

- Three resource counters
- Three action buttons
- Actions update multiple states
- Display student army power (sum)
- Prevent actions if insufficient resources

## Acceptance Criteria

- [ ] All resources tracked
- [ ] All actions work correctly
- [ ] Multiple states update together
- [ ] Validation prevents invalid actions
- [ ] Totals calculate correctly
