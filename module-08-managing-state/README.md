# Module 8: Managing Multiple State

## Story Context

Battle preparations intensify! Students must build complex dashboards managing multiple resources simultaneously - gold, mana, health potions, and troop deployments. Master coordinating multiple pieces of state.

## Learning Objectives

- Manage multiple related state variables
- Coordinate state updates
- Handle complex UI interactions
- Build tabbed interfaces
- Combine filtering, sorting, and pagination

## React Concepts

- Multiple useState hooks
- State coordination patterns
- Derived state vs stored state
- Complex event handlers
- Conditional rendering at scale

## Quests

1. **Resource Dashboard** - Track multiple counters
2. **House Tabs** - Tab navigation with state
3. **Creature Gallery** - Combined filter/sort/pagination
