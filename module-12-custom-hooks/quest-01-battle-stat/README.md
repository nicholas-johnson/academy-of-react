# Quest 1: useBattleStat Hook

Create hook managing battle stat (current value, max value) with increment, decrement, reset functions. Use for health, mana, stamina in battle interface.

## Requirements (TypeScript)
- Custom hook useBattleStat
- Takes initial and max values
- Returns value and functions
- Prevents going over max or below 0
- Use in component for 3 stats
- Generic TypeScript typing

## Acceptance Criteria
- [ ] Hook created in separate file
- [ ] All functions work correctly
- [ ] Boundaries enforced
- [ ] Used for multiple stats
- [ ] Properly typed with TS
- [ ] Reusable
