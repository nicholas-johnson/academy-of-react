# Quest 2: useSpellDelay Hook

Build useDebounce hook to prevent spell spam. User must wait 1 second between casts. Show cooldown timer.

## Requirements (TypeScript)
- Custom hook useDebounce or useSpellDelay
- Prevents action for X seconds
- Returns boolean (can cast?)
- Countdown timer
- Reset function
- TypeScript typing

## Acceptance Criteria
- [ ] Hook prevents spam correctly
- [ ] Timer counts down
- [ ] Boolean updates accurately
- [ ] Works with button disable
- [ ] Proper TypeScript types
- [ ] Reusable pattern
