# Module 7: Side Effects with useEffect

## Story Context

The Wizarding War approaches! Intelligence gathering begins. Students must master the useEffect hook to handle timers, fetch battle intelligence from external sources, and implement auto-save functionality.

## Learning Objectives

- Understand side effects in React
- Master useEffect hook
- Handle component lifecycle
- Implement cleanup functions
- Fetch data from APIs
- Work with timers and intervals
- Debounce operations

## React Concepts

- useEffect hook
- Dependency arrays
- Cleanup functions
- Async operations in effects
- Effect timing (mount, update, unmount)

## Setup

Navigate to `demo/`, run `npm install`, then `npm run dev`.

## Quests

1. **Meditation Timer** - Countdown timer with cleanup
2. **War Intelligence** - Fetch data from API
3. **Auto-Save** - Debounced localStorage sync
