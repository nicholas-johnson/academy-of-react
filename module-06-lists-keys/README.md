# Module 6: Lists, Keys, and Data Manipulation

## Story Context
House competitions intensify! Build ranking systems, leaderboards, and inventories. Master data manipulation with sorting, filtering, and pagination.

## Learning Objectives
- Understand the key prop and why it's critical
- Sort and filter arrays dynamically
- Implement pagination
- Build sortable tables
- Handle large datasets efficiently

## React Concepts
- Key prop for list items
- Conditional rendering
- Derived state
- Performance considerations with lists

## Setup
Navigate to `demo/`, run `npm install`, then `npm run dev`.

## Quests
1. **Battle Rankings** - Sortable table of 20 students
2. **Spell Inventory** - Filterable spell collection with multiple criteria
3. **Quest Pagination** - Paginated list of 50 quests
