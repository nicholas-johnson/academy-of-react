# Quest 1: Battle Rankings

Create a sortable table of 20 students with name, house, wins, losses, magicLevel. Click column headers to sort. Highlight top 3 students.

## Requirements
- 20 students in array
- Table with 5 columns
- Click headers to sort ascending/descending
- Visual indication of top 3
- Sort state tracked

## Acceptance Criteria
- [ ] 20 students displayed
- [ ] All columns sortable
- [ ] Sort direction toggles
- [ ] Top 3 highlighted
