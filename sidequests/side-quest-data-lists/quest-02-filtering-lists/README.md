# Quest 2: Spell Inventory

Display 30+ spells with multiple filter criteria: type (fire/ice/lightning/healing), power range (slider), mana cost range. Show filtered count.

## Requirements
- 30+ spells with full stats
- Type filter (checkboxes or select)
- Power range slider
- Mana cost range slider
- Display count of filtered results

## Acceptance Criteria
- [ ] 30+ spells in data
- [ ] All filters functional
- [ ] Filters work together
- [ ] Count updates
- [ ] Clear filters button
