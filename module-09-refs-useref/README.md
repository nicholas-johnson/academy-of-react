# Module 9: useRef and DOM References

## Story Context

**TypeScript Training Begins!** The Academy upgrades to TypeScript for type safety in battle systems. Learn useRef for DOM manipulation, focus management, and imperative operations.

## Learning Objectives

- Master useRef hook
- Access DOM elements directly
- Manage focus programmatically
- Trigger animations imperatively
- Understand ref vs state
- **NEW**: TypeScript with React

## React + TypeScript Concepts

- useRef with TypeScript
- Ref types (HTMLInputElement, HTMLDivElement, etc.)
- Type-safe props
- Interface definitions
- Generic types with hooks

## Setup

First time with TypeScript! Run `npm install` then `npm run dev`.

## Quests

1. **Battle Prompt** - Auto-focus input with refs
2. **Animation Trigger** - Trigger CSS animations via ref
3. **Replay Controls** - Video-like controls with refs
